# IPCC AR6 Synthesis Report 2023

- **ID:** SF, TA
- **Организация:** IPCC (Межправительственная группа экспертов по изменению климата)
- **Год:** 2023
- **URL:** https://www.ipcc.ch/report/ar6/syr/downloads/report/IPCC_AR6_SYR_FullVolume.pdf
- **Тип:** report
- **Надёжность:** high

## Описание

Синтетический отчёт Шестого оценочного доклада IPCC — итоговый документ крупнейшего научного обзора состояния климата Земли. Обобщает выводы трёх рабочих групп и специальных докладов цикла AR6. Включает данные о прогрессе в технологиях возобновляемой энергетики.

## Ключевые данные

- Стоимость солнечной энергии снизилась на 85% в период 2010–2019
- Стоимость ветровой энергии снизилась на 55% в период 2010–2019
- Стоимость аккумуляторов снизилась на 85% в период 2010–2019
- Научный консенсус о влиянии человека на изменение климата
- Оценка путей ограничения потепления на уровне 1.5°C и 2°C
- Анализ технологических решений для декарбонизации
