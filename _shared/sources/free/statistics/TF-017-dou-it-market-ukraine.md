# DOU.ua -- Аналітика IT-ринку України

- **ID:** TF-017
- **Організація:** DOU.ua (спільнота розробників України)
- **Рік:** ежегодно (останні дані -- 2025)
- **URL:** https://jobs.dou.ua/salaries/
- **Тип:** statistics / report
- **Надійність:** high

## Ключові дані

### Зарплати в IT (2024-2025)
- Середня зарплата розробника: ~$3,500/міс (зростання ~10% порівняно з попереднім роком)
- Спеціалісти в AI/Cloud: зростання зарплат на 15%+
- Senior Software Engineers (JS, Python, Java): $3,500-$5,000/міс
- Junior-позиції: зниження на ~5%
- Python розробники: ~$2,600/міс
- JavaScript (React, Angular): ~$2,400/міс
- Data Engineers: ~$3,840/міс (найбільше зростання серед data-спеціалістів)
- IT менеджери: ~$3,000/міс (зниження з $3,150)
- Тестувальники: ~$2,200/міс
- DevOps: медіана для Junior -- на $200 вище, ніж у розробників

### Топ-50 IT-компаній (січень 2025)
- Загальна кількість спеціалістів: 79,648
- EPAM Ukraine: 9,570 спеціалістів
- SoftServe: 7,279
- GlobalLogic: 5,660
- Ajax Systems: 4,160
- Luxoft: 3,200

### Топ-50 продуктових компаній (початок 2026)
- 24,500 технічних IT-спеціалістів

### Тренди ринку (2024-2025)
- Відтік за H2 2024: -858 спеціалістів (найменший з початку повномасштабного вторгнення)
- 16 компаній збільшили кількість спеціалістів у H2 2024
- Лідери зростання: Ajax Systems (+749), Nova Digital (+323), WIX (+150)
- 87% IT-спеціалістів надають перевагу віддаленій роботі (грудень 2024)
- Плани на 2025: 6 компаній планують найняти 200-500 нових спеціалістів, 17 компаній -- 50-200

## Методологія

DOU.ua проводить регулярні опитування серед IT-спеціалістів України. Дані про зарплати збираються через анонімні опитування двічі на рік (зима/літо). Рейтинг компаній базується на кількості спеціалістів за даними самих компаній.
